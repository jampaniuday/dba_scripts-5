Contains classes providing access to many String utilities as well as other String convenience classes.

Keep in mind that when using links to methods in this application, there are several conventions that should be implemented:

  • The following icons should be used for XML files located on the website:
    - XML Files
    - DTD Files
    - XSL Files
  • All source code should include doc comments that following standards documented in "DBA Coding and Documentation Standards" guidelines located within DBA Documentation Library.
  • All changes must be approved by the DBA Group before entering the production web server.
