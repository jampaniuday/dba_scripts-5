I. INTRODUCTION

  It is important to note that facilities for describing 
  regular expressions did not appear in Java 2 until Version 1.4.

  This is surprising given the power of regular expressions 
  in scripting languages and how ubiquitous they are 
  on the UNIX platform (where Java was first designed for).

  I provide two examples in this Java Programming Category:

    - RegDemoApache.java  (Using Apache Jakarta RegExp)
    - RegDemoSun.java     (Using Java / util.regexp from SDK 1.4)
